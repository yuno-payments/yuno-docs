The **Installments** section in the Yuno dashboard allows you to manage payment plans for your merchants without needing to use the API. You can view existing installments, create new ones, and edit or delete them as needed.

## Overview

Access the Installments section to monitor and configure the installment plans available for your payments.

  src="/images/using-yuno/dashboard-overview/installments-list.png"
  alt="Installments List"
/>

## Key Features

The dedicated Installments section provides several management capabilities:

*   **View installments list**: See all configured installment plans at a glance.
*   **Open installment details**: Access specific information for any individual installment.
*   **Search and filter**: Quickly find installments using the search bar or by applying filters (e.g., by name).
*   **Create a new installment**: Set up a new installment plan directly from the dashboard.
*   **Edit or delete**: Modify existing plans or remove those that are no longer needed.

## Installment Details

When viewing or creating an installment, the following fields are available:

| Field | Description |
| :--- | :--- |
| **Date Status** | The current status and timing of the installment plan. |
| **Name** | Identifying name for the installment plan. |
| **Merchant reference** | Your internal reference for the merchant associated with this plan. |
| **Country** | The country where the installment plan is applicable. |
| **Brand** | The card brand (e.g., Visa, Mastercard). |
| **Issuer** | The card issuer bank. |
| **BIN** | Bank Identification Number range for the cards. |
| **First deferred** | Details regarding the first payment deferral, if applicable. |
| **Currency** | The currency used for the installment values. |
| **Min value** | The minimum transaction value required for this installment plan. |
| **Max value** | The maximum transaction value allowed for this installment plan. |

## How to Create an Installment

1.  Navigate to the **Installments** section in the dashboard.
2.  Click on **Create installment**.
3.  Fill in the required information, including the merchant reference, country, and specific installment values.
4.  Save the plan to make it active.
