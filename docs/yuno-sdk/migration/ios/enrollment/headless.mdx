title: "Headless enrollment"
description: "Migrate iOS Headless enrollment (Yuno.apiClientEnroll) to the unified Transaction API."
---

Headless enrollment was a pure-tokenization API: you collected the data in your own UI, called `continueEnrollment(data:)`, and got back a dictionary with `vaultedToken`, `customerSession`, and `status`.

In the unified API, headless enrollment is no longer a separate client. It's the same `Transaction.start(paymentSelected:)` with `transactionData` set on the `PaymentMethodSelected`. The SDK runs the enrollment internally and reports completion through `onStatus`.

| Before | After |
|---|---|
| `Yuno.apiClientEnroll(countryCode:customerSession:)` | `SdkPayments.Transaction(config:)` |
| `EnrollmentCollectedData` / `CollectedData` / `CardData` | `SdkPayments.PaymentMethodSelected` + `SdkPayments.TransactionData` + `SdkPayments.CardTransactionData` |
| `continueEnrollment(data:)` | `start(paymentSelected:)` with `transactionData` |
| Result map with `vaultedToken` | `onStatus(status:)` (token retrieved from your backend) |

## Side-by-side

<CodeGroup>

```swift Before. Headless enrollment
import YunoSDK

Yuno.initialize(apiKey: "your-public-api-key")

let enrollHeadless = try Yuno.apiClientEnroll(
    countryCode: "CO",
    customerSession: "session-from-backend"
)

let data = EnrollmentCollectedData(
    customerSession: "session-from-backend",
    paymentMethod: CollectedData(
        type: "CARD",
        vaultedToken: nil,
        card: CardData(/* number, securityCode, expiration, holderName */),
        customer: nil
    )
)

let result = try await enrollHeadless.continueEnrollment(data: data)
let vaultedToken = result["vaultedToken"]
let status = result["status"]
```

```swift After. Unified Transaction
import SdkPayments

SdkPayments.initialize(apiKey: "your-public-api-key")

final class EnrollmentModel: SdkPayments.TransactionController {

    func onStatus(status: SdkPayments.TransactionStatus) {
        // On a successful status, retrieve the vaulted token from your
        // backend's "list customer payment methods" endpoint.
    }

    func enroll(card: SdkPayments.CardTransactionData) {
        let paymentSelected = SdkPayments.PaymentMethodSelected(
            paymentMethodType: "CARD",
            transactionData: SdkPayments.TransactionData(card: card)
        )

        let config = SdkPayments.TransactionConfig(
            customerSession: "session-from-backend",
            countryCode: "CO",
            controller: self
        )

        SdkPayments.Transaction(config: config)
            .start(paymentSelected: paymentSelected)
    }
}
```

</CodeGroup>

<Warning>
**The vaulted token is no longer returned to the client.** Headless enrollment's `continueEnrollment(...)` returned `[String: String]` containing `vaultedToken`. The unified API does not surface the token to the client. Fetch it from your backend's list-customer-payment-methods endpoint after the success status arrives in `onStatus`.
</Warning>

## Field mapping

| Before (Headless) | After (unified) | Notes |
|---|---|---|
| `Yuno.apiClientEnroll(countryCode:customerSession:)` | `SdkPayments.Transaction(config:)` | The transaction object replaces the headless client. |
| `EnrollmentCollectedData.customerSession` | `SdkPayments.TransactionConfig.customerSession` | Moves to config. |
| `CollectedData.type` | `SdkPayments.PaymentMethodSelected.paymentMethodType` | Moves to the parent object. |
| `CollectedData.card` (`CardData`) | `SdkPayments.TransactionData.card` (`SdkPayments.CardTransactionData`) | Same fields, namespaced under `transactionData`. |
| `CollectedData.customer` | `SdkPayments.TransactionData.customer` | Moves under `transactionData`. |
| `continueEnrollment(data:)` returning `[String: String]` | `start(paymentSelected:)` + `onStatus(status:)` | The SDK reports completion through `onStatus`. |

## Behavior changes

- **No `vaultedToken` on the client.** Fetch it from your backend after success.
- **`createPayment` is not invoked** during enrollment.
- **`vaultedToken` on `PaymentMethodSelected` is ignored.** Enrollment exists to produce a vaulted token; there's nothing linked yet to reference.
