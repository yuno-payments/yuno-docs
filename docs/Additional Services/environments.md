Yuno uses two environments—**Sandbox** (Test Mode) and **Production** (Live Mode)—for both the API and the dashboard. They share the same concepts but are documented in two places depending on how you use them:

* **API and integration:** Base URLs, credentials, timeouts, and how to call the Yuno API in each environment → [API Environments](https://docs.y.uno/reference/api-environments).
* **Dashboard:** How to switch between Test Mode and Live Mode in the Yuno dashboard, activation, and the Test mode toggle → [Environments (Dashboard)](https://docs.y.uno/docs/environments).

Use the link above that matches what you need (API reference or dashboard guide).
