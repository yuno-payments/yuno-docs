<Tip>
**Recommended SDK**

We recommend using the [iOS Seamless SDK](/docs/sdks/full-checkout/ios-payments) for enrollment.
</Tip>

## Step 1: Adopt YunoEnrollmentDelegate

```swift
protocol YunoEnrollmentDelegate: AnyObject {
    var customerSession: String { get }
    var countryCode: String { get }
    var viewController: UIViewController? { get }

    func yunoEnrollmentResult(_ result: Yuno.Result)
}
```

## Step 2: Start the enrollment flow

```swift
Yuno.enrollPayment(with: self, showPaymentStatus: true)
```

### Possible status values

* `success`
* `fail`
* `reject`
* `processing`
* `internalError`
* `userCancell`

## Step 3: Render mode (Advanced)

For full UI control, use `startEnrollmentRenderFlow`:

```swift
let enrollmentFlow = await Yuno.startEnrollmentRenderFlow(with: self)
let formView = await enrollmentFlow.formView(with: self)
```
