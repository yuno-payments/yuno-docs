Aida AI is integrated throughout the Yuno Dashboard, providing contextual assistance and AI-powered features directly within different dashboard sections. Aida AI helps you get instant answers, generate insights, and streamline your workflow without leaving the dashboard.

## Using Aida AI across the dashboard

Aida AI lives in one chatbot widget (bottom right). Open it on any screen, choose **Aida AI** or **customer support**, and ask for help in natural language.

### Dashboard entry point

The Aida AI widget is always available in the bottom-right corner of every dashboard screen, so you can access help wherever you are.

### Start a new chat

When you open the widget, you'll be prompted to start a new chat with Aida AI.

### Payments

On the Payments section, use the chat to ask about payment IDs, status, or recent failures.

### Insights

In the Insights section, ask Aida AI for metric explanations or quick charts.

<Image border={false} src="https://files.readme.io/c8069a9ff12a056ca040066e5ae42b5f3f9e7fff5eed8b381c6c27c1e80efa4c-aida-insights-chatbot.png" />

### Routing

Within Routing, ask Aida AI to create, optimise, or troubleshoot routing rules.

<Image border={false} src="https://files.readme.io/005216c766572cefdec6c8c512557ef9e445d6225c6d28a24b18e7139b7cd7b2-aida-routing-chatbot-open.png" />

How to use:

1. Open the chatbot widget in the bottom right.
2. Select **Aida AI**.
3. Ask your question.

## What you can ask Aida AI

Aida AI can help you with:

* Payment information: get details about payments, transactions, and history
* Insights and analytics: request explanations, charts, or emailed visualizations
* Routing configuration: get help creating and optimizing routing rules
* Platform features: learn about Yuno features, integrations, and capabilities
* Troubleshooting: resolve issues and understand error messages
* Technical questions: ask about SDK integration, API usage, and platform functionality

For more information about Aida AI's capabilities, see the [Aida AI Agent documentation](/docs/aida-ai-agent).
