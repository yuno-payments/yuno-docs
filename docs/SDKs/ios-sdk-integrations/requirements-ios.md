To implement the Yuno iOS SDK, you need to meet the following requirements:

* **Package Manager**: Add [CocoaPods](https://guides.cocoapods.org/using/using-cocoapods.html) or [Swift Package Manager](https://www.swift.org/package-manager/) to your project
* **iOS Version**: Use iOS version 14.0 or above
