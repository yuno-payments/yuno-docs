# Web SDK

All notable changes to the Yuno Web SDK will be documented in this page.

## [1.6.8] - 2026-04-15

<ChangelogBadge type="added" /> **unMountSdk() Helper**: Introduced a new top-level method for explicit SDK cleanup.
<ChangelogBadge type="changed" /> **Rollback Cancel Flow**: Unified cancel flow events. The SDK now explicitly reports `CANCELED_BY_USER` in the `yunoPaymentResult` callback.
<ChangelogBadge type="changed" /> **Legacy Event Rollback**: Legacy cancel-related events have been consolidated into the standard result flow.
<ChangelogBadge type="fixed" /> **Forter Token Listener**: Added a listener for the `ftr:tokenReady` event to ensure reliable capture of the Forter session token.

### Apple Pay Enhancements
<ChangelogBadge type="added" /> **Metadata Support**: The cancel flow now includes a `metadata` parameter with a `paymentCreated` boolean.
<ChangelogBadge type="added" /> **Contact Info Passthrough**: Automatically collects and forwards customer's email, phone, and name from `shippingContact`.

### PayPal REDIRECT Workflow
<ChangelogBadge type="added" /> **No OTT Flow**: Support for a `REDIRECT` workflow that skips OTT creation.
<ChangelogBadge type="added" /> **PaypalButtonModal**: New component that renders the PayPal button inside a modal.

## [1.6.0] - 2026-03-20

<ChangelogBadge type="added" /> **One-Click Payments**: Added support for one-click checkout with saved cards.
<ChangelogBadge type="fixed" /> **3DS Modal Centering**: Fixed an issue where the 3DS challenge modal was not centered on mobile devices.
