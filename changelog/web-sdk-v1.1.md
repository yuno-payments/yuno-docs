# Web SDK v1.1 Changelog

This changelog documents the changes, new features, and deprecated functionality in Web SDK version 1.1.

## What's New in v1.1

### New Features
- **Enhanced 3DS support**: Improved 3D Secure authentication flow
- **Performance improvements**: Better loading times and overall performance
- **Security enhancements**: Additional security measures for payment processing
- **User experience improvements**: Better UI/UX for payment forms

### Language Support
- **Expanded language options**: Added support for additional languages in payment forms
- **Language options available**: es (Spanish), en (English), pt (Portuguese), fil (Filipino), id (Indonesian), ms (Malay), th (Thai)

### Technical Improvements
- **Better error handling**: Improved error messages and handling
- **Enhanced debugging**: Better debugging capabilities for developers
- **Code optimization**: Optimized code for better performance

## What's Deprecated in v1.1

No features were deprecated in v1.1.

## Migration Guide

### From v1.0 to v1.1

1. **No breaking changes**: v1.1 is backward compatible with v1.0
2. **Optional upgrades**: You can take advantage of new features without changing existing code
3. **Performance benefits**: Automatic performance improvements without code changes
