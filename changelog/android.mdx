# Android SDK

All notable changes to the Yuno Android SDK will be documented in this page.

## [2.13.4] - 2026-04-20

<ChangelogBadge type="changed" /> **Font Family Customization**: Apply merchant custom `fontFamily` from `YunoConfig.styles` across all SDK components.

## [2.13.3] - 2026-04-05

<ChangelogBadge type="changed" /> **WebView State**: WebView loading state polish for better UX.

## [2.13.0] - 2026-02-15

<ChangelogBadge type="added" /> **Keeploader Support**: Support for Keeploader integration.

## [2.11.0] - 2025-11-10

<ChangelogBadge type="breaking" /> **CheckoutBuilder Migration**: Removed `cardFlow` from `YunoConfig`. Card flow configuration is now handled exclusively through the **CheckoutBuilder**.
> See the [Migration Guide](/changelog/migration-guides/android/v1-1-to-v1-2) for details.

---

## [1.24.0] - 2024-08-15

<ChangelogBadge type="added" /> **MODO APM**: MODO APM via deeplink flow support.
<ChangelogBadge type="changed" /> **Mercado Pago**: Checkout Pro now opens in external browser (WebView deprecated).
