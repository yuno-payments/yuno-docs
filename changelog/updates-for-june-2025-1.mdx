## New fields added

* Within `additional_data` -> `airline`

| Field Name | Type | Description  | Examples                                        |
| ---------- | ---- | ------------ | ----------------------------------------------- |
| `type`     | enum | Type of trip | `ONE_WAY`, `ROUNDTRIP`, `MULTIPLE_DESTINATIONS` |

* Within `additional_data` -> `legs`

| Field            | Type | Description                                                                                                                                                                                                                                                                                                                                                                                                                                      | Example |
| ---------------- | ---- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------- |
| **order**        | int  | A running index (starts with 1), describing the order of legs in the same route.                                                                                                                                                                                                                                                                                                                                                                 | 2       |
| **route\_order** | int  | A running index (starts with 1), describing the order of routes by time.<br /><br />E.g: If an order contains 2 Routes:<br />New-York→London→Paris (connection in London)<br />• New-York→London should have route\_order=1, order=1<br />• London→Paris should have route\_order=1, order=2<br /><br />Paris→London→New-York<br />• Paris→London should have route\_order=2, order=1<br />• London→New-York should have route\_order=2, order=2 | 1       |
