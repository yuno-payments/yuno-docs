User management provides control over who can access your B2B organization and what actions they can perform. Access is governed by role-based permissions assigned at both the Account and Account Group levels, allowing for precise security management.

### Key capabilities

- **Identity Management**: Create and maintain user profiles with standard contact information.
- **Granular Permissions**: List, update, and remove permissions for specific accounts or entire account groups.
- **Batch Operations**: Use the replace functionality to perform a full reset of a user's permissions in a single request.
