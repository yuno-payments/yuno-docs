Account groups are used to organize your accounts into logical sets. This is particularly useful for large organizations with multiple business units or partners.

### Create Account Group
Use this endpoint to create a new organization group.

[api:create-account-group]

### List Account Groups
Retrieve a paginated list of all account groups in your organization.

[api:list-account-groups]

### Get Account Group
Retrieve details for a specific account group.

[api:get-account-group]

### Update Account Group
Update the name or merchant ID of an existing account group.

[api:update-account-group]
