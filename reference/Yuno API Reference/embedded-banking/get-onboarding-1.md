This endpoint retrieves detailed information about a specific onboarding for a recipient in the embedded banking solution. Use this endpoint to check the current status, configuration, and details of an existing onboarding.

This endpoint requires an existing onboarding created via [Create Onboarding](ref:create-onboarding).
